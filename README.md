UUID issues for Burp Suite
==========================

Building
--------

Execute `ant`, and you'll have the plugin ready in `burp-uuid.jar`

Dependencies
------------

 - JDK 1.8+ (tested on OpenJDK 8, Debian/Ubuntu package: `openjdk-8-jdk`)
 - Apache ANT (Debian/Ubuntu package: `ant`)
